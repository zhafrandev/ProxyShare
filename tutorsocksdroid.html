Tutorial SocksDroid
Untuk memasang IP Proxy menggunakan aplikasi SocksDroid, ikuti langkah-langkah dibawah ini.
Unduh dan Install SocksDroid
Anda dapat mengunduh aplikasi SocksDroid dari Google Play Store dan menginstalnya di perangkat Android Anda.
Buka Aplikasi SocksDroid
Setelah Anda menginstal aplikasi, buka SocksDroid di perangkat Anda.
Konfigurasi Proxy
Di layar utama SocksDroid, ikuti langkah-langkah berikut:
✓ Ketuk ikon "Tombol Pengaturan" (biasanya ikon roda gigi) di pojok kanan atas layar.
✓ Lalu pilih "Add Profile" dan buatlah nama yang anda inginkan lalu klik oke.
✓ Lalu pada "Server IP", masukan IP yang proxy yang kamu dapatkan.
✓ Pada "Server Port", masukan nomor port yang kamu dapatkan juga.
✓ Langkah terakhir, Aktifkan tombol di bagian atas aplikasi untuk mengaktifkan SocksDroid.
✓ Selesai.
